==============
django-ezutils
==============


Overview
---------

Django-ezutils consists of the following things:

    - Common base abstract models for inherit.

    - Common fields, forms, functions for use.

Installation
-------------

- Add ``django_ezutils`` to your Django settings ``INSTALLED_APPS``::

    INSTALLED_APPS = [
        # ...
        "django_ezutils",
    ]
